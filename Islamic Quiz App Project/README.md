# Islamic Knowledge Quiz App

A responsive, interactive web application designed to test and improve knowledge about Islam. Built with HTML, CSS, Bootstrap 5, and JavaScript.

Project Overview
This project was developed as part of a Web Technologies assignment to demonstrate proficiency in:
Core web technologies (HTML/CSS/JS)
DOM manipulation and Event Handling 
Data persistence using `localStorage`
Responsive UI design with Bootstrap 

Features
Three Categories: Pillars of Islam, Prophets, and Quran Knowledge.
Interactive Quiz Engine: Immediate feedback (Green/Red) and explanations for every answer.
Progress Tracking: Live progress bar and question counter.
High Scores: Saves your best scores locally so they persist after refreshing.
Responsive Design: Works seamlessly on mobile, tablet, and desktop.
